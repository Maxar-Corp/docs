About These Docs

You're viewing documentation for the Gruntwork Infrastructure Packages.

An Infrastructure Package is a reusable, best-practices definition of a piece of infrastructure written mostly in Terraform, with some Golang and Bash where needed.

The code for an Infrastructure Package is stored in a private GitHub repo made available to Gruntwork clients. These docs were auto-generated from those private GitHub repos.

The source code for an Infrastructure Package is private, but the documentation files cover foundational DevOps concepts and AWS best practices that are helpful to many users.

We hope you enjoy them!
